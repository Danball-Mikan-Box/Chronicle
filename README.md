# Chronicle — 物語を紡ぐ、静かな執筆環境

**Chronicle** は、小説を書くことに特化したデスクトップ執筆支援ツールです。

Markdown の記法を知らなくても、書式ボタンを押すだけで太字や見出し、ルビが使えます。
書いた内容は右ペインに即座にプレビューされ、まるで Obsidian のようなライブ編集体験を提供します。

## コンセプト

### 1. 「書くこと」に集中する

ツールの操作ではなく、物語そのものに向き合う時間を大切にします。
エディタは余計な装飾を排し、原稿用紙のような落ち着いた背景色と書体で構成されています。
書式ツールバーは必要に応じて使い、あとはひたすら書き進めるだけです。

### 2. プロジェクトで物語を管理する

一つの作品を「プロジェクト」として管理します。
章ごとにファイルが分割され、サイドバーからいつでも切り替えられます。
フォルダ構成はシンプルで、`.md` ファイルとして保存されるため、他のツールとの連携も容易です。

```
my-novel/
├── chronicle.json    # プロジェクト設定
└── chapters/
    ├── 01-序幕.md
    ├── 02-第一章.md
    └── 03-第二章.md
```

### 3. Markdown を知らなくても書ける

書式ツールバーのボタンを押すと、エディタに Markdown 記法が自動挿入されます。
「これってどう書くんだっけ？」と迷うことなく、直感的に文章を装飾できます。
ルビ（ふりがな）もボタン一つで挿入できます。

### 4. 縦書きにも横書きにも

ワンクリックで縦書き・横書きを切り替えられます。
縦書き時は CSS の `writing-mode: vertical-rl` により、プレビューが縦組で表示されます。
和文小説は縦書きで、WEB投稿用は横書きで、といった使い分けが可能です。

### 5. 執筆のモチベーションを保つ

リアルタイムの文字数カウント、目標文字数に対する進捗バー、読了時間の目安を常に表示。
「今日はあと何文字書くか」という目標を視覚化し、執筆のペースメーカーとして機能します。

## 機能一覧

- **プロジェクト管理**: 作品ごとにプロジェクトを作成、章を追加/削除
- **Markdown エディタ**: シンタックス不要の書式ツールバー（太字/斜体/見出し/引用/リスト/リンク/ルビ）
- **ライブプレビュー**: 右ペインに即時反映
- **縦書き/横書き切替**: ボタン一つで切り替え
- **ルビ対応**: `{漢字|かんじ}` → ルビ付きテキスト
- **文字数カウント**: 空白除く文字数、目標進捗バー、読了時間
- **ファイル保存**: プロジェクト構成は JSON、本文は Markdown として保存
- **自動保存**: 3秒の無入力後に自動保存（ステータスバーに状態表示）
- **キーボードショートカット**: Ctrl+S 保存、Ctrl+B 太字、Ctrl+I 斜体
- **ダークモード**: ツールバーのボタンでワンタッチ切替
- **章名の変更**: ダブルクリックで章名を編集
- **最近のプロジェクト**: サイドバーから履歴を開く
- **エクスポート**: 単一ファイルまたは全章を TXT/HTML で出力
- **ステータスバー**: ファイル名、保存状態、文字数、自動保存状態を表示

## 技術スタック

- **言語・フレームワーク**: Rust + Dioxus 0.6 (Desktop)
- **Markdown 処理**: pulldown-cmark
- **ファイルダイアログ**: rfd (Rust File Dialog)
- **レンダリング**: wry (WebView)
- **データ保存**: JSON (プロジェクト構成) + Markdown (本文)

## セットアップ

```bash
# リポジトリをクローン
git clone <url>
cd chronicle

# ビルドして実行
cargo run
```

## 使い方

1. **「新規」** ボタンからプロジェクトを作成
2. **「+ 章を追加」** で章を追加
3. エディタに本文を書き、**書式ツールバー** で装飾
4. **「保存」** ボタン、**Ctrl+S**、または自動保存（3秒無入力後）
5. **「開く」** で既存プロジェクトを読み込み
6. 章名を**ダブルクリック**で変更
7. **「出力」** で TXT または HTML として書き出し

### ショートカット
| キー | 機能 |
|------|------|
| Ctrl+S | 保存 |
| Ctrl+B | 太字 |
| Ctrl+I | 斜体 |
