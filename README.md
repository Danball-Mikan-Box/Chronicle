# Chronicle — 物語を紡ぐ、静かな執筆環境

**Chronicle** は、小説を書くことに特化した執筆支援ツールです。
Markdown を知らなくても書式ボタンで装飾でき、書いた内容は即座にプレビューされます。

---

## 特徴

- **プロジェクト管理**: 作品ごとにプロジェクトを作成。章・話・資料をツリー管理
- **Markdown エディタ**: 書式ツールバー（太字/斜体/見出し/引用/リスト/リンク/ルビ）＋ショートカット
- **ライブプレビュー**: 右ペインまたは下ペインに即時反映。縦書き/横書き切替対応
- **ルビ対応**: `{漢字|かんじ}` 記法でルビ付きテキスト
- **文字数カウント**: リアルタイムカウント、目標進捗バー、読了時間目安
- **自動保存**: 3秒無入力後に自動保存（キャンセルセーフ）
- **ダークモード**: ワンタッチ切替。画面全体に適用
- **集中モード**: サイドバー・プレビューを隠してエディタに没頭
- **パネルカスタマイズ**: サイドバー・エディタ・プレビューの表示/非表示とリサイズ
- **フォント・レイアウト設定**: フォント種類、サイズ、行間、最大幅、字下げを設定可能
- **資料管理**: キャラクター・世界観・用語集などカテゴリ別に資料を保存
- **タブ編集**: 複数の話や資料をタブで切り替え

## 出力形式

| 形式 | 内容 |
|------|------|
| プロジェクトバックアップ (ZIP) | 全ファイルを圧縮。ファイル名は `{作品名}-backup.zip` |
| サイト出力 (ZIP) | HTMLサイト形式。index.html + 話ごとのページ。モバイル対応、作者名表示 |
| 章・話ごとに分割 (ZIP) | テキスト (.txt) または HTML (.html) で出力 |
| **投稿サイト用 (ZIP)** | ルビ記法を各サイトの書式に変換して出力 |

### 投稿サイト対応

| サイト | ルビ記法 |
|--------|---------|
| 小説家になろう | `｜漢字《かんじ》` |
| カクヨム | `漢字《かんじ》` |
| ハーメルン | `\|漢字《かんじ》` |

各話を `.txt` ファイルとして章フォルダに格納し、ZIPにまとめて出力します。

## 技術スタック

- **言語・フレームワーク**: Rust + Dioxus 0.6 (Desktop / Mobile)
- **Markdown 処理**: pulldown-cmark
- **ファイルダイアログ**: rfd (Rust File Dialog)
- **WebView**: wry / WebKitGTK
- **データ保存**: JSON (プロジェクト構成) + Markdown (本文)
- **パッケージング**: cargo-deb / cargo-generate-rpm / AppImage / cargo-wix

## セットアップ

```bash
# ビルドして実行（デスクトップ）
cargo run --release

# Android ビルド（要 dioxus-cli + Android SDK）
rustup target add aarch64-linux-android
cargo install dioxus-cli
dx build --platform android --release
```

### 依存パッケージ（Linux）

```bash
sudo apt install libgtk-3-dev libwebkit2gtk-4.1-dev libjavascriptcoregtk-4.1-dev \
  libcairo2-dev libpango1.0-dev libgdk-pixbuf-2.0-dev libglib2.0-dev librsvg2-dev
```

## 使い方

1. **「新規」** でプロジェクトを作成
2. サイドバーの **「+」** で章や話を追加
3. エディタに書き、書式ツールバーで装飾
4. **Ctrl+S** または **「保存」** で手動保存（3秒無入力で自動保存）
5. **「出力」** から各種形式で書き出し
6. 章名・話名・資料名は **✎** ボタンまたはダブルクリックで変更

### ショートカット

| キー | 機能 |
|------|------|
| Ctrl+S | 保存 |
| Ctrl+B | 太字 |
| Ctrl+I | 斜体 |

## プロジェクト構成

```
my-novel/
├── chronicle.json        # プロジェクト設定（作品名・作者・書字方向など）
├── chapters/
│   ├── 01-第一章/
│   │   ├── 01-第一話.md
│   │   └── 02-第二話.md
│   └── 02-第二章/
│       └── 01-第一話.md
└── materials/
    ├── 01-主人公.md
    └── 02-年表.md
```

|----------------|--------|
| Linux | `.deb` / `.rpm` / `.tar.gz` / `.AppImage` |
